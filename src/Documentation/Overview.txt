Uebersicht
==========

Koch-Suite besteht aus folgenden Bestandteilen:

 - eine Reihe von Kommandozeilen-Tools:
   - mmimport: Import von Meal-Master-formatierten Rezepten
   - mmexport: Export von        ---       "      ----
   - rezept2latex: Ausgabe von Rezepten als LaTeX-Source.

 - ein paar WWW-Seiten
   - index.php: Haupt-Menue
     - Such-Funktion
   - rezept.php: Ausgabe des formatierten Rezeptes
   - print.php: ausgabe des Rezeptes als reiner Ascii-Text
