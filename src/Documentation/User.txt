User-Dokumentation
==================

Installations-anleitungen sind im Top-Level-Verzeichnis dieses
Programmpaketes enthalten. Der Zweck dieses Dokuments ist eine
Dokumentation der vorhandenen Kommandozeilen-Tools.

mmimport
--------
  mmimport ließt von der Standard-Eingabe und parsed die Rezepte.
  Es ist nicht notwendig, dass die Eingabe nur aus einem Rezept besteht.
  Der Zweck dieses Tools ist eine einfache Schnittstelle zur Datenbank,
  die durch Pipes bequem verwendet werden kann. Speziell eignet sich 
  mmimport um Usenet-Artikel (z.B. der Gruppe de.rec.mampf) aus z.B. slrn 
  heraus einzulesen.

mmexport
--------
  mmexport wird mit einem Suchstring als Parameter aufgerufen. Es
  durchsucht die Datenbank nach einem Menue, dass im Titel den String
  enthaelt. Bei der Suche wird die Gross- und Kleinschreibung nicht
  berücksichtigt.
  Die Ausgabe geschieht auf die Standard-Ausgabe.
  Das Rezept wird in einem Meal-Master-kompatibeln Format ausgegeben.
